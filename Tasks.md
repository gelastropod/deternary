# Tasks

### 0. Introductory

In the following tasks, whenever an integer $N$ is outputted, $N \\,\\%\\, 19683$ should be outputted instead.

- Task 0: Cat program  
You are given an integer $N$, $0 \le N < 19683$. Output $N$.

  Input: $N$  
Output: $N$

- Task 1: Addition  
You are given two integers $A$ and $B$, $0 \le A, B < 19683$. Output $A + B$.

  Input: $A, B$  
Output: $A + B$

- Task 2: Maximum  
You are given two integers $A$ and $B$, $0 \le A, B < 19683$. Output $\max(A, B)$.

  Input: $A, B$  
Output: $\max(A, B)$

- Task 3: Arithmetic  
You are given an integer $N$, $0 \le N < 4$. You are then given two integers $A$ and $B$, $0 \le A, B < 19683$. Based on $N$, perform the following operation on $A$ and $B$:
  - $N = 0$: $C = A + B$
  - $N = 1$: $C = A - B$
  - $N = 2$: $C = A \cdot B$
  - $N = 3$: $C = \left\lfloor\frac{A}{B}\right\rfloor$

  Output $C$.

  Input: $N, A, B$  
Output: $C$
